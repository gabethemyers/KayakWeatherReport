# Kayak Weather Report Monorepo
This repository contains the frontend and backend API for a marine weather reporting app created for kayak guides on the Elkhorn Slough in Moss Landing, CA.

** Screen shot of frontend **

## Project Structure
- `/kayak-weather/`: The React frontend
- `/kayak-weather-api/`: The Go backend 

## Tech Stack
**Frontend**
- React
- TypeScript
- Vite
- Tailwindcss

**Backend**
- Go
- Chi
